# king-mineo-extension

マイネ王掲示板既読管理(Chrome拡張)

## 概要(Overview) 

マイネ王掲示板に既読管理機能を加えるChrome用の拡張機能です。<br>
Chrome拡張機能が使えるのは、PC版のChromeと、PC版とAndroid版のFirefoxです。

## 導入方法(Setup)

### Chrome の場合

全てのファイルを任意のフォルダーにダウンロードします。

次に、Chrome の操作で拡張機能を有効にします。

<img src="https://github.com/user-attachments/assets/63ca18b0-89ac-45e7-9dfe-6f2626923418" width="40%">

1. Chrome のタブを開き、URLに「Chrome://extensions」と入力する  
  (もしくは、メニューの設定から「拡張機能」を選択して表示する)
  
2.  トグルボタンで「デベロッパー モード」にする

3. 「パッケージ化されていない拡張機能を読み込む」を選択  
 ファイルを保存したフォルダを選択する

正常に読み込まれると以下のように追加されます。

<img src="https://github.com/user-attachments/assets/bfc3cd64-614a-4927-80bb-88f990c8d8c8" width="30%">

### Firefox の場合(PC版)

署名済ファイル(.xpi)をダウンロードします。  
Firefoxの検索窓に「abouut:addons」と入力するか「設定」から「拡張機能とテーマ」を選択します。  
「ファイルからアドオンをインストール」を選択し、.xpi ファイルを選択します。

### Android版Firefoxの場合

署名済ファイル(.xpi)をダウンロードします。  
拡張子を .xpi->.zip に修正します。  
「設定」「Firefoxについて」で、情報画面へ遷移して、Firefoxの画像部分を6回以上タップします。  
「ファイルから拡張機能をインストール」が選択肢に表示さるのでタップしてダウンロードしたファイルを選択します。  
※ この操作は、開発者版のアプリ「Firefox Nightly」と同じですが、署名無しで読めるか読めないかの違いです。

## 操作方法(Usage)

特に必要な操作はありません。  
掲示板の詳細画面を開いた際に、既読として登録され、一覧画面を表示した際に、未読か既読かを判定して色を変えています。

掲示板の一覧画面では、必要に応じて、キーボードによる未読・既読の強制操作が可能です。<br>
操作したい掲示板にマウスを合わせ、キーを操作します。

| キー | 操作内容 |
----|---- 
| R | 未読・既読の切り替え |
| D | 非表示。"R"で再表示 |
| S | 情報保管 json形式 |
| L | 情報復元 |

Firefox版で、Android で使用する場合は、キー入力が出来ません。
代わりに、長押し操作で、状態を切り替える事が出来ます。

既読管理のデータは端末(ブラウザ)単位に保存されるので、複数端末で同期を取りたい場合には、  
ファイルへの保管と復元を行います。
基本的には、拡張機能の「オプション」画面で操作が出来ます。

オプション画面を表示するには「拡張機能の管理」アイコンから開けます。
